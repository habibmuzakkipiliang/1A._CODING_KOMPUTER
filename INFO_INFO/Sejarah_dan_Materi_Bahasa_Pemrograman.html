Sejarah dan Materi Bahasa Pemrograman

Pengertian HTML

HTML atau Hyper Text Markup Language adalah sebuah bahasa markah untuk membuat halaman web. Paham kan maksudnya ? Kalau belum paham, sini saya jelaskan… Jadi, HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. Kalau kita ibaratkan nih.....HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar. Bagi programmer web, HTML merupakan bahasa pertama yang wajib dikuasai.

Fungsi HTML

  1. Membangun struktur halaman website
  2. Menampilkan informasi di browser
  3. Menandai teks
  4. Menampilkan media
  5. Menampilkan Formulir

HTML juga merupakan dasar bagi pengembangan web, dan sering digunakan bersama dengan CSS (Cascading Style Sheets) untuk mengatur tampilan visual halaman web dan JavaScript untuk menambahkan interaktivitas

Sejarah HTML

Penemu HTML (Hyper Text Markup Language) adalah Tim Berners-Lee, tepatnya pada tahun 1991 namun belum dirilis secara resmi saat itu. Ide tentang HTML bermula pada saat dia bekerja di CERN, Laboratorium Eropa untuk Fisika Partikel di Jenewa, Swiss. Saat itu para peneliti fisika sering berkolaborasi dan melibatkan lembaga dari seluruh dunia. Tim Berners-Lee memiliki ide agar peneliti dari seluruh dunia mampu mengumpulkan informasi dengan mudah, sejak saat itu, Tim Berners-Lee disebut Bapak Website Modern.

Pengertian CSS

CSS merupakan singkatan dari Cascading Style Sheets. CSS adalah bahasa yang digunakan untuk mengatur tampilan visual dari sebuah halaman web. Dengan CSS kita dapat mengatur warna, ukuran, tata letak dan lain sebagainya.

Jika diibaratkan sebagai bangunan maka CSS adalah tampilan dari bangunan tersebut seperti warna cat pada dinding, dekorasi hingga pencahayaannya. Sedangkan HTML diibaratkan sebagai fondasi dan struktur bangunan tersebut seperti dinding, lantai dan atapnya. Tanpa CSS maka tampilan dari halaman web sangat sederhana dan tidak menarik. Seperti gambar di bawah ini!

Fungsi-fungsi utama CSS

  1. Mengatur Tampilan Website:
  2. Membuat Desain Halaman Website
  3. Membuat Website Responsif (bisa disesuaikan dengan berbagai Perangkat)
  4. Mengontrol Layout Website
  5. Mengatur Animasi dan Transisi Website
  6. Memudahkan Pengelolaan Kode

Sejarah CSS

Sejarah CSS (Cascading Style Sheets) dimulai saat pertama kali diusulkan oleh Hakon Wium Lie pada 10 Oktober 1994 yang saat itu bekerja dengan Tim Berners-Lee (bapak HTML) di CERN. Organisasi Riset Nuklir Eropa dikenal sebagai CERN. Sejak itu, Hakon Wium Lie dikenal sebagai Bapak CSS.

Pengertian JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk membuat halaman web interaktif dan dinamis. Bahasa ini dijalankan oleh browser web dan memungkinkan pengguna untuk menambahkan fitur seperti animasi, validasi formulir, dan interaksi dengan API. JavaScript sering dipasangkan dengan HTML dan CSS untuk menciptakan pengalaman pengguna yang lebih menarik dan responsif.

Fungsi utama JavaScript

  1. Membuat halaman website interaktif
  2. Manipulasi DOM (Document Object Model)
  3. Validasi Formulir HTML
  4. Pengembangan aplikasi web dengan React, Vue, dan Angulatr
  5. Animasi dan efek visual Website
  6. Pembuatan Game Web
  7. Pengembangan aplikasi seluler dengan React Native dan Ionic
Tambahan :

Website interaktif dalam JavaScript adalah website yang menggunakan JavaScript untuk membuat elemen-elemen web merespon tindakan pengguna, seperti klik tombol, mengisi formulir, atau menggerakkan mouse. Dengan JavaScript, website menjadi lebih dinamis dan responsif, memberikan pengalaman pengguna yang lebih menarik dan fungsional.

Sejarah JavaScript

JavaScript pertama kali diciptakan oleh Brendan Eich, seorang karyawan Netscape, pada tahun 1995. Netscape kala itu merupakan perusahaan software ternama yang dikenal dengan web browser miliknya, Netscape Navigator.

Ibarat dan Perumpamaan

🏠Ibarat Rumah:
  1. HTML (Hyper Text Markup Language)

    2. ➤ Struktur atau kerangka Rumah: HTML itu seperti kerangka bangunan. Dia menentukan apa saja yang ada: pintu, jendela, atap, dinding. Misalnya: paragraf, heading, tabel, gambar, form.

  2. CSS (Cascading Style Sheet)

    3. ➤ Desain atau cat dan dekorasi rumah : CSS itu seperti kamu ngecat tembok, pasang gorden, furnitur, bikin rumah kelihatan cantik. CSS menentukan tampilan: warna, ukuran, jarak, posisi, dll.

  3. JavaScript
    4. ➤ Listrik dan mesin otomatis di rumah : JavaScript membuat rumah hidup dan interaktif. Kayak saklar lampu otomatis, pintu buka-tutup otomatis, atau AC nyala saat suhu panas. Di website, JavaScript bikin tombol bisa diklik, form bisa validasi otomatis, dan konten bisa berubah tanpa reload halaman.
🧍‍♂️ Ibarat Manusia:
  1. HTML = Tulang dan organ

    2. ➤ Memberikan struktur dasar tubuh: tangan, kaki, kepala.

  2. CSS = Pakaian dan penampilan

    3. ➤ Membuat manusia kelihatan menarik: baju, gaya rambut, warna kulit.

  3. JavaScript = Otak dan gerakan

    4. ➤ Memberikan fungsi dan interaksi, seperti bergerak, bicara, berpikir.

Pengertian Python

Python adalah bahasa pemrograman yang banyak digunakan dalam aplikasi web, pengembangan perangkat lunak, ilmu data, Artificial Intellegent (AI), machine learning (ML). Developer menggunakan Python karena efisien dan mudah dipelajari serta dapat dijalankan di berbagai platform.

Fungsi utama Python

  1. Pengembangan Web
  2. Analisis Data
  3. Kecerdasan Buatan (AI) dan Machine Learning (ML)
  4. Scripting dan Otomatisasi
  5. Pengembangan Aplikasi Desktop
  6. Pengembangan Game
  7. Testing Software
  8. Tes Perangkat Lunak

Sejarah Python

Python dibuat pada akhir 1980-an oleh Guido van Rossum di Centrum Wiskunde & Informatica (CWI) di Belanda sebagai penerus bahasa ABC (sendiri terinspirasi oleh SETL), mampu menangani pengecualian dan berinteraksi dengan sistem operasi Amoeba. Implementasinya dimulai pada bulan Desember 1989.

Pengertian Java

Java adalah bahasa pemrograman berorientasi objek dan multiplatform yang populer digunakan untuk mengembangkan berbagai aplikasi, mulai dari aplikasi desktop, aplikasi web, aplikasi mobile, hingga aplikasi embedded.

Fungsi Utama Java

  1. Pengembangan Aplikasi Mobile (Android)
  2. Pengembangan Aplikasi Desktop (Windows dan Linux)
  3. Pengembangan Aplikasi Web
  4. Pengembangan Aplikasi Lintas Platform

Sejarah Java

Java pertama kali dibuat oleh James Gosling & para penemu dari Sun Microsystem yang tergabung dalam Green Project tahun 1991-1992. James Golsing & teman2 nya berhasil membuat sebuah program dan dinamakan OAK. Nama Oak, diambil dari pohon oak yang tumbuh di depan jendela ruangan kerja "bapak java", James Gosling. Tapi sayangnya nama OAK sudah dipakai oleh perusahaan Oak Technology.

Karena itu, mereka pun harus memikirkan beberapa nama untuk program tersebut, diantarannya Silk, DNA, dan JAVA. Belum diketahui siapakah yang mengusulkan nama JAVA tersebut. Tapi kabarnya nama JAVA itu diambil dari nama tiga develover kuncinya: James Gosling, Arthur van hooff, dan Andy Bechtolsheim.

Ada kabar lain yang dikira paling benar tentang penamaan JAVA ini. Yaitu ketika James Gosling & teman2 nya sedang pergi ke Indonesia, tepatnya di pulau Jawa. Mereka memikirkan nama baru untuk program buatannya tersebut. Dan saat James Gosling dan para rekannya meminum segelas secangkir kopi panas yang berasal dari Jawa, Indonesia. James Gosling pun terinspirasi dengan secangkir kopi panas yang diminumnya tersebut. Apalagi James Gosling dan rekannya tersebut adalah pecinta kopi. Tanpa pikir panjang lagi, mereka pun langsung memberi nama program buatannya dengan nama JAVA yang terinspirasi dengan Kopi Jawa (Java Coffee) dari negara tercinta kita Indonesia. Dan JAVA pun diluncurkan ke publik pada tahun 1995 dengan Logo yang berbentuk sebuah Kopi Panas kesukaan James Gosling sendiri.
