Cerita selama naik Kereta Api Diesel dan KRL Commuter Line ke Palmerah Jakarta

Catatan : KRL Commuter Line adalah kereta rel listrik yang melayani perjalanan dari Rangkasbitung menuju wilayah Jabodetabek (Jakarta, Bogor, Depok, Tangerang, dan Bekasi). Sementara itu, kereta api diesel (KRD) adalah kereta api berbahan bakar diesel yang melayani perjalanan dari Rangkasbitung menuju Walantaka (Petir), Serang , Cilegon, dan Merak.

Pada Hari Sabtu, 23 Agustus 2025. Saya dan ayah berangkat dari rumah ke Stasiun Kereta Api Serang pada pukul 05.15 WIB, dengan cara jalan kaki ke sana. dan kami tiba di Stasiun Kereta Api Serang pada pukul 05.35 WIB. Dan kami tunggu di Stasiun Kereta Api Serang sampai pukul 06.00 WIB, karena kereta api yang saya tumpangi berangkat pada pukul 06.00 WIB. Dan kereta api yang saya tumpangi adalah Kereta Api Diesel (KRD) jurusan Serang - Rangkasbitung lalu dilanjutkan ke Jurusan Tanah Abang. Dan kereta api ini tiba di Rangkasbitung pada pukul 07.00 WIB. Dan selama perjalanan dari Stasiun Serang ke Stasiun Rangkasbitung, saya melihat pemandangan yang indah di sepanjang perjalanan, seperti sawah, perumahan, dan lain-lain.

Dan dilanjutkan dengan Jurusan Rangkasbitung - Tanah Abang dengan menggunakan KRL Commuter Line jurusan Rangkasbitung - Tanah Abang. Dan KRL ini tiba di Stasiun Tanah Abang pada pukul 09.00 WIB. Dan selama perjalanan dari Stasiun Rangkasbitung ke Stasiun Tanah Abang, saya melihat pemandangan yang indah di sepanjang perjalanan, seperti perumahan, gedung-gedung tinggi, dan lain-lain, setelah sampai di Pasar Tanah Abang Blok C, saya pun makan di Rumah Makan Lapau Jambua Aia khas Bukittinggi yaitu dengan Nasi Goreng untuk saya dan Rendang dan Sayur Nangka untuk ayah saya.

Setelah makan di Pasar Tanah Abang Blok C, saya dan ayah pergi ke Palmerah Jakarta dengan naik KRL Commuter Line lagi, dan kami tiba di Palmerah pada pukul 09.49 WIB, kami tunggu di Halte Stasiun Palmerah sampai pukul 10.10 WIB, dan yang anterin saya adalah Om Nan dengan menggunakan motor, sementara ayah order motor Maxim sampai ke rumah nya, dan kami pun sampai di rumah pada pukul 10.30 WIB.

Sampai disana, saya dan ayah saya pun ngobrol puas dengan Om Nan dan istrinya sampai siang pukul 12.00 WIB, lalu dilanjutkan dengan makan siang dan sholat dzuhur, dan pada pukul 13.00 sampai 13.30 WIB, kami pun istirahat sejenak sebelum melanjutkan perjalanan pulang ke Kota Serang.

Pada pukul 13.45, saya pun bersiap-siap untuk pulang ke Kota Serang. Kami pun berangkat dari rumah Om Nan dan menuju Stasiun Palmerah untuk naik KRL Commuter Line. Dan kami tiba di Stasiun Palmerah pada pukul 13.55 WIB, dan kami tunggu di Stasiun Palmerah sampai pukul 14.00 WIB, karena KRL Commuter Line jurusan Tanah Abang - Rangkasbitung berangkat pada pukul 14.10 WIB. Dan KRL ini tiba di Stasiun Rangkasbitung pada pukul 15.32 WIB. Dan selama perjalanan dari Stasiun Palmerah ke Stasiun Rangkasbitung, saya melihat pemandangan yang indah di sepanjang perjalanan, seperti gedung-gedung tinggi, perumahan, dan lain-lain.

Di Rangkasbitung, dan kami pun istirahat yaitu Sholat Ashar dan sedikit makan Nasi Goreng Lapau Jambua Aia. Setelah istirahat, kami pun ganti kereta lagi dengan menggunakan Kereta Api Diesel (KRD) jurusan Rangkasbitung - Serang. Dan kereta api ini tiba di Stasiun Serang pada pukul 17.40 WIB. Dan selama perjalanan dari Stasiun Rangkasbitung ke Stasiun Serang, saya melihat pemandangan yang indah di sepanjang perjalanan, seperti sawah, perumahan, dan lain-lain, saya pun sampai di rumah pada pukul 18.00 WIB.
